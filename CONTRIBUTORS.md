This repo is maintained under **NeuroGamingLab** and **OpenClaw** for commit attribution. The repository’s local Git config is set to use NeuroGamingLab (`user.name` / `user.email`) so that commits from this project stay consistent. To keep it that way after cloning, run:

```bash
git config user.name "NeuroGamingLab"
git config user.email "NeuroGamingLab@users.noreply.github.com"
```
